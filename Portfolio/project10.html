About Project

Project done by: Adobe Illustrator
Method: Design
This is a creation of a representative name card, this is to showcase that I am a graphic designer, and from the whole aspect is more to gaming, and technology vibe , displaying a infocomm student, and my hobby of playing games.

Logo

My logo is a game console since I am a game designer i used that idea. I want my company to fly higher and better each year so i added wings on both sides of the game console, thinking that i am very interested and love game, i tried to create a heart shape over the game console and add stars above to show my interest. I also add 4 console buttons at the back of the namecard to make it look more like a designer.

Color Scheme

I used the color harmony of double split complementary because it is an intricate color scheme offers a great deal of color contrast while still blending harmoniously. I have a black background to contrast all the other color which allow audience to see it clearly. In a whole, trying to protray neon-looking namecard. There is a mini jagged white background behind the logo as to make the logo look more outstanding.

Typography

For the informations like website,phone,etc is used Broadway font For the name is used Gill Sans Ultra Bold, cause name is important to let people know so i used this font to make it bold and thick in looking.For the designation is used Bradley Hand ITC so as to differenciate the font between the name and designation, allow audience to see both infomation at once. For the phrase ‘WE MAKE YOU PLAY’ and the website i chose Comic Sans MS as it is simple and clear to see.

Back Face Of NameCard

Back view include a mini version logo, my name, my role, , and all my contact details and social medias so people can contact me easily. There is also some input of my own drawing and using illustrator to create some characters. The color used here, is still the same as the font design. There is icon infront of the contact details , to make it more visually looking and alsi to help those who cannot read words.

Sketches

I created many mock up sketches before deciding on the final one. I chose this in the end, is because The logo being emphasize as the biggest object in the whole namecard , the informations and name plus logo is very important so fonts and size is slightly bigger and bold. It is visual appearance in symmetry and have equality of weight between opposing shapes/objects. No additional other color used other then the main pink blue and green. Color Harmony is used too. There is unity used in my poster as it is all being brought together by using consistency color and fonts
